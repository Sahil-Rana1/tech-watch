# TechWatch Electronics - E-Commerce Website

A modern, professional, and fully responsive Electronics E-Commerce website built with HTML, CSS, JavaScript, and Bootstrap 5.

## Features
- **Modern Tech-Style UI:** Sleek, dark/light mode, and premium aesthetics.
- **Responsive Design:** Mobile, tablet, and desktop optimized.
- **Dynamic Catalog:** Product listing with filtering (Brand, Price, Category) and sorting.
- **Product Details:** High-quality image zoom, detailed specifications, and related products.
- **Cart & Wishlist:** Fully functional LocalStorage-based cart and wishlist.
- **User System:** Simulated login, registration, and order history.
- **Admin Dashboard:** Simulated dashboard with sales analytics (Chart.js) and product management.
- **Smooth Animations:** Hover effects, fade-ins, and loading skeleton screens.

## How to Run
Since this project uses **ES Modules** (`type="module"`) and dynamic imports, browsers will block script execution if the files are opened directly (via `file://`).

**To run the website locally:**

1. **Using VS Code (Recommended):**
   - Open the project folder in VS Code.
   - Install the **"Live Server"** extension.
   - Click **"Go Live"** in the bottom right corner or right-click `index.html` and select "Open with Live Server".

2. **Using Node.js:**
   - Open a terminal in the project directory.
   - Run `npx serve`.
   - Open the provided URL in your browser.
